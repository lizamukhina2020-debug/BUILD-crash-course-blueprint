# BUILD blueprint (standalone)

- **Open in browser:** [`BUILD_BLUEPRINT.html`](./BUILD_BLUEPRINT.html) (double-click in Finder, or right-click → Open With → Safari/Chrome).
- **One zip to share:** [`BUILD_BLUEPRINT_bundle.zip`](./BUILD_BLUEPRINT_bundle.zip) — contains the HTML + `blueprint-media/` (all four images). Send this file in DMs or email; recipients unzip and open the HTML.
- **Not bundled** with the SeedMind mobile app unless you deliberately embed or link it.

**Where on your Mac:** `Desktop/SeedMind/docs/` — open that folder in Finder (`Cmd`+`Shift`+`G` in Finder, paste the path).

**Export as PDF:** Open the HTML → **File → Print → Save as PDF**.

**Images:** Included in `blueprint-media/` — see [`blueprint-media/README.md`](./blueprint-media/README.md) for filenames.
